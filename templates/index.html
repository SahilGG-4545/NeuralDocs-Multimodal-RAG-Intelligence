Capabilities

Built for Intelligence

A production-ready multimodal RAG pipeline combining vision and language.

👁️
CLIP Multimodal Embeddings
OpenAI CLIP encodes text and images into the same 512-d vector space for unified semantic search.
🔍
Two-Stage Retrieval
FAISS ANN search followed by cosine similarity reranking extracts the most relevant chunks.
🦙
Llama 4 Scout
Meta's Llama 4 Scout 17B–16E MoE model via Groq delivers fast, high-quality multimodal answers.
📄
Full PDF Extraction
PyMuPDF extracts chunked text and embedded raster images for complete document coverage.
⚡
Smart Query Classification
Automatically detects visual vs. text queries and adjusts retrieval strategy for best results.
📊
Latency Telemetry
Every response includes embedding, retrieval and LLM timings for full pipeline observability.
